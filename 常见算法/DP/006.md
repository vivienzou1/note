### **爬楼梯**
#### **问题描述**
有N个阶梯的楼梯，每次只能爬一个或者两个台阶，求爬到楼梯顶端有多少种爬法？
**JS实现**

	function stairs(N) {
		// body...
		var aux = [];
		aux[0] = 1;
		aux[1] = 1;
		for (let i = 2; i <= N; i++) {
			aux[i] = aux[i-1] + aux[i-2];
		}
		return aux[N];
	}
注：

1、对于第i阶台阶，考虑到他的前一步，只有两种情况：要么是从第i-1阶过来，要么是从第i-2阶过来，除此外没有其他的来源。
